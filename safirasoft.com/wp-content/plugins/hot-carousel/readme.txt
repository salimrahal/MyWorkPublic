== Description ==

"HOT Wordpress Carousel" plugin by [hotWP](http://www.hotjoomlatemplates.com/free-wordpress-plugins "hotWP") is a fully configurable, simple but powerful carousel plugin, based on jQuery. It will help you to show your photos in a carousel style.

== Installation ==

1. Install plugin as usually from WordPress.org, or download, unpack and upload `hot_carousel` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Publish Hot Carousel widget in one of available widget sidebars.
4. Make sure that Load jQuery is enabled only if your theme doesn't load it.
5. Enter "Folder where your images are stored" link, in example http://www.yoursite.com/images/folder
6. In the widget options, you can see many parameters that defines the widget's look. Experiment with these to configure the widget for your site.
